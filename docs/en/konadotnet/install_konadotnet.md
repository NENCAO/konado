# Installation

## Basic Dependencies

1. Install the Konado plugin (required)
2. A Godot version that supports C# (4.5+ recommended)

## Installation Steps

1. Extract the konadotnet plugin into the `addons` directory of your Godot project
2. Build the project and make sure there are no errors
3. In the Godot editor, go to `Project -> Project Settings -> Plugins` and enable the Konado and Konadotnet plugins
